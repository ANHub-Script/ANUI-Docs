Installation

Copy and paste this script into your executor to load the latest version of ANUI.

Lua 
local ANUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/ANHub-Script/ANUI/refs/heads/main/dist/main.lua"))()

Profile Configuration Recommended

Define a global profile variable to reuse across tabs easily.

Lua 
-- Define Base Profile
local BaseProfile = {
    Banner = "https://...",
    Avatar = "rbxassetid://...",
    Status = true,
    Badges = { ... }
}

-- Helper Function
local function MakeProfile(data)
    local p = {}
    for k,v in pairs(BaseProfile) do p[k]=v end
    if data then for k,v in pairs(data) do p[k]=v end end
    return p
end

Window

The main container for your interface.

Lua
local Window = ANUI:CreateWindow({
    Title = "Script Hub",
    Author = "Username",
    Icon = "rbxassetid://...",
    Theme = "Dark",
    Transparent = false 
})
PropertyTypeDescription
TitlestringWindow title text.
AuthorstringSub-text below the title.
ThemestringPreset themes ("Dark", "Light", "Rose", etc).

Tab

Creates a new menu category in the sidebar.

Lua
local MainTab = Window:Tab({
    Title = "Home",
    Icon = "home",
    Profile = MakeProfile({
        Title = "Welcome User",
        Desc = "Main Dashboard"
    })
})

Button

A clickable element that executes a callback function.

Lua
MainTab:Button({
    Title = "Click Me",
    Desc = "This is a description",
    Icon = "zap",
    Callback = function()
        print("Clicked!")
    end
})

Toggle

A switch to turn features on or off.

Lua
MainTab:Toggle({
    Title = "Auto Farm",
    Value = false,
    Callback = function(state)
        print("AutoFarm:", state)
    end
})
